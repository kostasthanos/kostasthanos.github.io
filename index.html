Projects

Number System Converter


A program for converting a number from one system to another. Available systems : Binary (bin), Decimal (dec), Octal (oct) and Hexadecimal (hex).

Social Network in C++


A simple example of a Social Network made in C++. Basic functions such as : make a post, see wall, send friend request, accept/reject a friend request and a few more.

Easter Date


A project (research) on finding Easter's date both for Orthodox and Catholic using Gauss's algorithm with an implementation in Python.

Dice Roller Game


A two player game with dices made with HTML, CSS and JS. If a player rolls two 1's or two 6's then he loses his turn and the score he has won in the current round.

Tkinter Calculator


Scientific calculator made with Python and Tkinter library. Contains functions such as abs, mod, factorial, trigonometric functions, roots, logarithms and some others.
