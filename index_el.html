Εργασίες - Εφαρμογές

Number System Converter


Πρόγραμμα μετατροπής ενός αριθμού από ένα σύστημα σε ένα άλλο. Διαθέσιμα συστήματα : Δυαδικό (bin), Δεκαδικό (dec), Οκταδικό (oct) και Δεκαεξαδικό (hex). Κατασκευή σε γλώσσα Python.

Social Network in C++


Ένα πολύ βασικό κοινωνικό δίκτυο σε γλώσσα C++. "Ποσταρίσματα", ο τοίχος μου, αποστολή αιτήματος φιλίας, αποδοχή ή όχι αιτημάτων φιλίας, αποτελούν κάποιες από τις βασικές λειτουργίες του προγράμματος.

Easter Date


Υπολογισμός της ημερομηνίας που "πέφτει" το Πάσχα για οποιασδήποτε χρονιά, τόσο των Ορθοδόξων όσο και των Καθολικών χρησιμοποιώντας τον αλγόριθμο του Gauss. Υλοποίηση σε γλώσσα Python.

Dice Roller Game


Παιχνίδι - εφαρμογή ζαριών με δύο παίκτες φταγμένο με HTML, CSS και Javascript. Αν ένας παίκτης φέρει διπλό 1 ή διπλό 6, τότε χάνει τη σειρά του και το σκόρ που έχει κερδίσει μέχρι τότε στο συγκεκριμένο γύρο.

Tkinter Calculator


Επιστημονική αριθμομηχανή. Υλοποίηση σε γλώσσα Python με Tkinter. Περιέχει βασικές συναρτήσεις όπως απόλυτη τιμή, παραγοντικό, τριγωνομετρικές συναρτήσεις, ρίζες, λογαρίθμους κ.α..

Covid-19 Greece Data and Plots


Hand Detection and Finger Counting


Εντοπισμός χρώματος και υπολογισμός-μέτρηση πλήθους δακτύλων με χρήση κάμερας. Ανάλυση όρων όπως contours, convex hulls, defect points κ.α.. Υλοποίηση σε Python με OpenCV.

Virtual on Screen Keyboard


Εικονικό πληκτρολόγιο χρησιμοποιώντας κάμερα. Ο χρήστης μπορεί να πληκτρολογήσει οτιδήποτε θέλει και να το αποθηκεύσει σε ηλεκτρονικό αρχείο. Χρήση γλώσσας Python και OpenCV.

Image Viewer with OpenCV


Παρουσίαση φωτογραφιών από φάκελο με εντοπισμό χρώματος. Ο χρήστης μπορεί να πλοηγηθεί σε επόμενη ή προηγούμενη φωτογραφία με κίνηση του χρώματος (αντικειμένου) δεξιά ή αριστερά αντίστοιχα.

Drowsiness Detection


Ανίχνευση υπνηλίας σε πραγματικό χρόνο. Πραγματοποιούνται δύο έλεγχοι. Ένας για την ανίχνευση κλεισίματος ματιών και ένας που αφορά την ανίχνευση χασμουρητών.

Virtual Drawing


Spot the Differences


Εντοπισμός διαφορών μεταξύ 2 εικόνων. Υλοποίηση με absdiff της βιβλιοθήκης OpenCV αλλά και με το πακέτο skimage. Ο χρήστης επιλέγει τις 2 εικόνες που θέλει και το πρόγραμμα εντοπίζει τις μεταξύ τους διαφορές.
